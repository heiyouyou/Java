Java数组

  • 数组是多个相同类型数据的组合，实现对这些数据的统一管理。
  • 数组中的元素可以是任何数据类型，包括基本数据类型和引用数据类型。
  • 数组属引用类型，数组型数据是对象(object)，数组中的每个元素相当于该对象的成员变量。

一、一维数组声明式

  • type var[] 或 type[] var；如：int arr[]int[] arrString[] arr
  • Java语言中声明数组时不能指定其长度(数组中元素的数)如：int a[5];非法表达式

二、一维数组初始化

  • 动态初始化：数组声明且为数组元素分配空间与赋值的操作分开进行。
  • 分配空间：int[] arr = new int[3];
  • 赋值：arr[0] = 3;arr[1] = 9;arr[2] = 8;
  • 静态初始化：在定义数组的同时就为数组元素分配空间并赋值
  • int a[] = new int[]{3,9,8};或者int[] a = {3,9,8};后者只有一维数组可以这样子表示
  • 注意：动态初始化一定要写维度（长度），而静态从初始化不需要写维度

三、数组元素的引用

  • 定义并用运算符new为之分配空间后，才可以引用数组中的每个元素；
  • 数组元素的引用方式：数组名[数组元素下标]
  • 数组元素下标可以是整型常量或整型表达式。如：a[3]，b[i]，c[6*i]；
  • 数组元素下标从0开始；长度为n的数组合法下标取值范围: 0 —>n-1，如：int a[] = new int[3]；可引用的数组元素为a[0]、a[1]、a[2]
  • 每个数组都有一个属性length指明它的长度，例如：a.length指明数组a的长度(元素个数)，不是a.length()
  • 数组一旦初始化，其长度是不可变的。

四、二维数组

  • 格式1（动态初始化）：int[][] arr = new int[3][2];

    1.定义了名称为arr的二维数组

    2. ~[3][2]表示：二维数组中有3个一维数组，arr[i]就是指向一维数组的内存地址

    3.~[2]表示：每一个一维数组中有2个元素

    4.一维数组的名称分别为arr[0], arr[1], arr[2]

    5. 给第一个一维数组下标为1的位置赋值为78写法是：arr[0][1] = 78；

  • 格式2（动态初始化）：int[][] arr = new int[3][];

    1.arr[i][]表示：每个一维数组都是默认初始化值null(注意：区别于以上格式1）如：System.out.printl(arr[2][1])//报错：java.lang.NullPointerException（空指针异常）

    2.可以对这个三个一维数组分别进行初始化：arr[0] = new int[3]；arr[1] = new int[1]；arr[2] = new int[2]；注：arr[1] = {1,4,5,6};//报错int[][]arr = new int[][3];//非法

  • 格式3（静态初始化）：int[][] arr = new int[][]{{3,8,2},{2,7},{9,0,1,6}};也可以这样：int arr1[][] = {{1,5,6},{2,2},{4,6,7}};

    1.定义一个名称为arr的二维数组，二维数组中有三个一维数组

    2.每一个一维数组中具体元素也都已初始化

    3.第一个一维数组表示方式：arr[0] = {3,8,2};

    4.第二个一维数组表示方式：arr[1] = {2,7};

    5.第三个一维数组表示方式：arr[2] = {9,0,1,6};

    4.第i个一维数组的长度表示方式：第三个一维数组的长度表示方式：arr[i].length;

  • 注意特殊写法情况：int[] x,y[]; x是一维数组，y是二维数组。
